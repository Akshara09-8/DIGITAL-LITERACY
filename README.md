# Digital Literacy Project

## Project Overview
This project focuses on developing essential **digital literacy skills** for students, including professional online presence, coding practice, email etiquette, and cybercrime awareness. The project consists of five tasks:  

1. **Task 1 – Presentation:** Created an infographic on digital literacy, tools, and safe internet practices using Canva.  
2. **Task 2 – Portfolio:** Set up profiles on LinkedIn, GitHub, and Kaggle, documenting skills and professional presence.  
3. **Task 3 – Platforms:** Practiced coding on HackerRank and created a Google Form for digital literacy awareness.  
4. **Task 4 – Email Etiquette:** Drafted sample emails and developed a checklist for professional communication.  
5. **Task 5 – Cybercrime Awareness:** Researched cybercrime prevention and created a “Stay Safe Online” checklist.
